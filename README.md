# Студия создания шортсов

> Мощная студия для создания Shorts/Reels/TikTok из длинных видео: AI-автонарезка, удаление неречевых участков, стильные субтитры в стиле CapCut и пакетная обработка.

## 🚀 Что это за проект

**ShortsCreatorStudio** — это развитие идеи [VideoCaptioner](https://github.com/WEIFENG2333/VideoCaptioner):

- база: генерация субтитров для видео;
- улучшения: расширенные эффекты субтитров, гибкие настройки и визуальный монтажный workflow;
- главное нововведение: **Auto Shorts** — модуль автоматической нарезки видео на шортсы с помощью ИИ.

Проект ориентирован на создателей контента, которым нужно быстро получать пачки вертикальных клипов с готовыми субтитрами и красивой подачей.

---

## ✨ Ключевые возможности

### 1) Auto Shorts (ИИ-автонарезка)

- автоматический поиск интересных моментов по ASR/семантике;
- фильтр повторяющихся моментов (анти-дубли);
- вырезание/сжатие неречевых сегментов внутри клипа;
- тонкая настройка пауз и отступов, чтобы не резать слова на полуслове;
- выбор диапазона анализа и ручной отбор кандидатов перед рендером.

### 2) Визуальный шаблон монтажа

- интерактивные зоны **WEBCAM/GAME** (crop + раскладка в вертикальном кадре);
- realtime-предпросмотр композиции;
- цветокоррекция каждого слоя (яркость/контраст/насыщенность/резкость);
- сохранение шаблона между перезапусками.

### 3) Субтитры уровня CapCut-style

- стили, эффекты и расширенные параметры отображения;
- гибкая обработка текста и таймингов;
- поддержка пакетной генерации субтитров для множества шортсов.

### 4) Пакетная обработка

- массовая обработка файлов;
- режимы: транскрибация, субтитры, full process;
- настройки качества финального видео (FPS / разрешение / профиль качества).

---

## 🧠 Как устроен pipeline

1. Анализ аудио и речи (ASR)
2. Поиск кандидатов для шортсов
3. Анти-дубликаты + фильтрация
4. Рендер вертикальных клипов
5. Пакетная генерация/применение субтитров

---

## 🛠 Технологии

- **Python + PyQt5** — десктопный интерфейс
- **FFmpeg** — нарезка, монтаж и кодирование видео
- **ASR + LLM-интеграции** — анализ речи и поиск интересных фрагментов
- Конфигурируемая система параметров для тонкой настройки качества результата

## 📦 Быстрый старт (dev)

> Ниже базовый ориентир для запуска из исходников. В проекте могут быть дополнительные зависимости/настройки окружения.

1. Установить Python 3.10+
2. Установить FFmpeg и добавить в PATH
3. Клонировать репозиторий
4. Установить зависимости
5. Запустить приложение

```bash
git clone https://github.com/MechaniksLab/ShortsCreatorStudio.git
cd ShortsCreatorStudio
pip install -r requirements.txt
python main.py
```
